In _Determining Solar Cell Parameters and Degradation Rates from Power Production Data_, we demonstrated the exciting possibility of extracting the diode-model parameters and degradation rates of a photovoltaic system from commoditized power generation and weather data.

Link to the article: https://doi.org/10.1016/j.ecmx.2022.100270

Link to the open-access online database: https://gitlab.in2p3.fr/energy4climate/public/sirta-pv1-data
